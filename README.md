# Superstore Sales Analysis

## Overview
Exploratory data analysis of 9,994 US retail orders (2014–2017) using Python to identify profitability drivers, discount impact, regional performance gaps, and seasonal trends.

## Tools Used
- Python, pandas, matplotlib, seaborn
- Jupyter Notebook (VS Code)

## Key Findings
- **Furniture** has a profit margin of only 2.49% vs 17%+ for other categories
- **High discounts (40%+)** generate an average loss of $106.71 per order
- **Central region** underperforms all other regions on profit margin
- **Sales peak** in January, March, and October — suggesting corporate budget cycles drive demand

## Business Recommendations
- Reduce discounts above 20% across all categories
- Review Furniture pricing and cost structure
- Investigate cost drivers in the Central region

## Files
- `analysis.ipynb` — full analysis notebook with charts and findings
- `Sample - Superstore.csv` — dataset (9,994 orders)
